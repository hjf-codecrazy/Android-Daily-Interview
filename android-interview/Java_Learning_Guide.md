## 💡 高效记忆的 5 个技巧

### 技巧 1️⃣：关键词推导法
**步骤**：
```
看关键词 → 脑海推导答案 → 对照文字检查

例：看到"synchronized | volatile | Lock"
推导：这是关于线程安全的不同方案...
```

### 技巧 2️⃣：表格对比法
**方法**：
- 利用文件中的表格进行对比记忆
- 不要只记个别特点，要记住整体对比关系
- 示例：ArrayList vs LinkedList 的 6 个对比维度

### 技巧 3️⃣：代码复述法
**过程**：
```
1. 看代码，理解逻辑
2. 关闭代码，自己手写
3. 对照原代码检查
4. 解释每一行的作用
```

### 技巧 4️⃣：常见错误法
**应用**：
- 每个题目都有"常见错误"提示
- 记住错误的原因，避免面试时犯同样错误
- 如：String 池、equals() 和 == 的混淆

### 技巧 5️⃣：间隔重复法
**时间表**：
```
第 1 次学习 → 第 1 天复习 → 第 3 天复习 → 第 7 天复习 → 第 14 天复习
```
